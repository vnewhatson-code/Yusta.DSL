# Юста — ИИ-ассистент для написания документации

**Юста** — это интеллектуальный ассистент, созданный для ускорения и упрощения процесса написания документации. Он помогает специалистам, не обладающим специальными навыками технического писателя или аналитика, создавать качественную и структурированную документацию по их продуктам.

---

## 🎯 Зачем нужна Юста?

Написание документации — это часто рутинная и трудоёмкая задача, требующая специфических навыков. Юста берёт на себя основную работу:

- **Ускоряет** процесс создания документации
- **Упрощает** работу тем, кто не является профессиональным техническим писателем
- **Повышает качество** документов благодаря управляемости и предсказуемости результата

---

## ✨ Основные возможности

- **Генерация документации с нуля** — Юста создаёт документы на основе контекста, который предоставляет разработчик
- **Аудит существующей документации** — проверяет и улучшает уже написанные материалы
- **Подход Schema Guided Reasoning** — снижает количество галлюцинаций, делает результат управляемым и предсказуемым
- **Гибкий выбор модели** — используйте любую поддерживаемую LLM (через OpenRouter или напрямую)

---

## 🧠 Как это работает?

Юста использует подход **Schema Guided Reasoning** — структурированный вывод, который:

1. Минимизирует галлюцинации ИИ-модели
2. Обеспечивает предсказуемость и повторяемость результата
3. Позволяет управлять форматом и содержанием генерируемой документации

Ассистент полностью реализован на базе **Dify** — мощной no-code платформы для создания ИИ-приложений. Пайплайн Юсты собран из готовых узлов, что обеспечивает:

- Прозрачность логики работы
- Простоту модификации и кастомизации
- Надёжность и воспроизводимость результатов

## ⚙️ Настройка

### Подключение LLM

Юста поддерживает работу с любыми моделями, доступными через API. Рекомендуемый провайдер — **OpenRouter**, который предоставляет единый доступ к множеству моделей.

1. Получите API-ключ у выбранного провайдера (например, на [openrouter.ai](https://openrouter.ai/))
2. В интерфейсе Dify перейдите к настройкам узлов **LLM** внутри пайплайна Юсты
3. Укажите выбранного провайдера и вставьте ваш API-ключ
4. Выберите модель, которую хотите использовать

## 🚀 Использование

Для начала работы с Юстой просто **начните чат** в интерфейсе Dify.

Чем более подробный контекст вы предоставите, тем точнее и полезнее будет результат.
